# INTEGRAQA Website

This is the official website for INTEGRAQA, a company that provides professional web services and software testing.

## Description

The website showcases the services offered by INTEGRAQA, which include:
- Mobile App Testing
- API Testing
- Web Application Testing
- SaaS & Cloud Platform Testing
- CRM Testing
- Website Testing
- Website Maintenance
- Systems Integration Testing
- Desktop App Testing

## Technologies

The website is built using the following technologies:
- HTML
- CSS (TailwindCSS)
- JavaScript

## Pages

The website consists of the following pages:
- `index.html`: The main landing page.
- `about.html`: Information about the company.
- `contact.html`: Contact information and a contact form.
- `cookies-policy.html`: The cookies policy of the website.
- Various pages for each service offered.

## How to Use

To view the website, open the `index.html` file in your web browser.
